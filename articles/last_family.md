### Scraping on Yahoo Finance

Some functions rely on web scraping (Yahoo Finance webpages), such as
[`last_indices()`](https://clement-lvd.github.io/financr/reference/last_indices.md)
and
[`last_crypto()`](https://clement-lvd.github.io/financr/reference/last_crypto.md)
which respectively provide the latest financial values (in USD) for over
40 global market indices and 100 cryptocurrencies.

[`last_indices()`](https://clement-lvd.github.io/financr/reference/last_indices.md)
return a `data.frame` with the latest financial value of major world
financial indices.

``` r
indices <- last_indices()
str(indices)
```
